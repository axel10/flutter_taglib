## 0.0.1

* Initial release of flutter_taglib.
* Support reading and writing metadata tags (Title, Artist, Album, Genre, Year, Track Number, Comment) for MP3, FLAC, M4A, OGG, WAV, and more.
* Support cover art extraction, update, and removal.
* Support technical audio properties (Duration, Bitrate, Sample Rate, Channels, and CBR/VBR/Unknown bitrate mode detection).
* Support Android Scoped Storage via File Descriptors and automatic write permission request integration.
* Support selectable platform configuration build hook (disable/enable specific native platforms).
